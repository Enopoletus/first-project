How to edit the dislikes

1. Get curl working on your computer (if it's not already). If you're on Windows 10, install bash.exe by installing a Linux Distro in the Windows 10 store.

2. Open the Chrome Dev Tools to the Network tab. Dislike a post.

3. Copy the HTTP POST request via right-click-->Copy-->Copy as cUrl.

4. Copy that into the bash. Edit with arrow keys, backspace key, and keyboard at will.

5. The copied cUrl works as thus: %5C=\
%22="
%2C=,
%3A=:

6. The number of dislikes in a post is the number listed after {\"n\":"
